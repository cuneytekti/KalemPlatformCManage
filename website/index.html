Niyə Kalem Platform?

1989-dan bəri 2.000-dən çox layihədə formalaşan təcrübə — pərakəndə komandalarının real ehtiyaclarından doğulmuş vahid idarəetmə platforması.

🧩

ERP-dən asılı olmayan iş

Pərakəndə idarəetməniz ERP-yə bağlı qalmır: platforma tam müstəqil işləyir. İstəsəniz SAP, LOGO, Mikro, 1C, jRetail, Venüt, Microsoft Dynamics 365 kimi aparıcı ERP-lərlə dəyişkən datalar API üzərindən ikitərəfli sinxronlaşır.

🔐

Beynəlxalq standartlarda şifrələmə

Bütün məlumatlar beynəlxalq standartlarda şifrələnir; həssas bilgilər rol əsaslı gizlədilir. Hər müştəri tam izolyasiya olunmuş mühitdə saxlanır.

📜

Sertifikatlı e-Kassa tətbiqi

Azərbaycan e-kassa inteqrasiyası ilə sertifikatlaşdırılmış tətbiq: satış, qaytarma, X/Z hesabatları vergi tələblərinə tam uyğun, avtomatik fiskalizasiya ilə.

Modullar

Kassadan idarə heyətinə — pərakəndənin hər səviyyəsi üçün peşəkar modullar.

01

Satış və mağaza

POS Satış

Barkod-öncəlikli sürətli kassa: çəkili məhsullar, endirimlər, qarışıq ödənişlər, offline dayanıqlıq və bank terminalları ilə tam avtomatik ödəniş.

Back Office

Məhsul kataloqu, qiymət siyasəti, stok hərəkətləri, filial hesabatları və rol əsaslı istifadəçi idarəetməsi — brauzerdən.

Mobil Anbar Terminalı

Android əl terminalları ilə mal qəbulu, inventarizasiya, rəf etiketi və qiymət yoxlaması — anbar əməliyyatları cibinizdə.

02

Müştəri və gəlir

CRM — Müştəri Qazanma

360° müştəri profili, çoxkanallı loyallıq, seqmentasiya, kampaniya və avtomatlaşdırma — müştəri qazanmağa fokuslanmış CRM. Aşağıda ətraflı.

B2B və e-Faktura

Təchizat zənciri idarəetməsində B2B uyğunluğu: təchizatçılarla e-faktura inteqrasiyaları, sifariş-faktura uzlaşdırması.

03

Təchizat və istehsal

Satınalma

Müqavilə şərtlərinə bağlı gondol, rəf başı, istifadə sahəsi və erkən ödəniş primləri; qiymət fərqləri və xidmət fakturalarının avtomatik yaradılması. Satınalma komandanızın gücünə güc qatır.

İstehsal

Mağaza içi və mərkəzi istehsal (fırın, təzə məhsul): mamul–yarımmamul–xammal izləməsi, istehsal maliyyətləri və məhsuldarlıq analizi.

Demirbaş Satınalma və Tender

Demirbaş alışları üçün tender idarəetməsi: tender elanı, şərtnamə hazırlanması, təkliflərin dəyərləndirilməsi və satınalma addımları — hamısı təsdiq axınlarına bağlı şəffaf süreç.

04

Əməliyyat və proses

Süreç İdarəetməsi

Qiymət dəyişikliyindən kampaniyaya, SKT nəzarətindən sayımlara — bütün əməliyyatlar mobil təsdiq axınlarına bağlı. Ofisdə olmadan idarə edin.

Mağaza Layihə İdarəetməsi

Mağaza hələ açılmadan: memarlıq cizgisindən AI dəstəkli yerləşim planı, müqavilələrdəki rəf payı vədlərinə uyğun kateqoriya bölgüsü və təchizatçılara avtomatik ilk sifarişlər.

ServiceDesk — Daxili Xidmətlər

ServiceDesk əsaslı ticket idarəetmə sistemi: mağaza əsaslı təmir, təmirat və dəyişdirmə işləri, mərkəzi anbardan sərf materialı sifarişləri — davamlı, izlənə bilən süreç axınları ilə.

05

Analitika və süni intellekt

KBoss — AI İdarəetmə

Şirkət sahibləri üçün süni intellekt dəstəkli idarəetmə: səsli əmrlə hesabat hazırlanması, tapşırıq idarəetməsi və anlıq KPI-lar.

AI Mağaza Əməliyyatları

Süni intellekt dəstəkli sifariş və sayım tövsiyələri, sifarişlərin təchizatçılara avtomatik göndərilməsi, sifariş-nəzarətli mal qəbulu ilə təchizatçı performansının ölçülməsi.

BI və Hesabat

Bold BI, Power BI və Microsoft 365 məhsulları ilə birbaşa inteqrasiya: mövcud hesabat mədəniyyətinizi dəyişmədən dərin analitika.

Dövri Analiz — AI Menecer Köməkçisi

Dövri analiz şirkətdəki maliyyə süreçlərinin idarəetməsini asanlaşdırır. Qərar alma, idarəetmə və sürətli saha əməliyyatları üçün süni intellekt dəstəkli menecer köməkçisi modulu.

CRM Modulu

Müştəri qazanmağa fokuslanmış CRM

Müştərini tanıyın, seqmentləşdirin, düzgün kampaniya ilə qayıdışa çevirin. Kalem CRM satış nöqtəsindən mobil tətbiqə qədər bütün kanalları tək müştəri görünüşündə birləşdirir.

  • 360° müştəri profili — bütün kanallardan alış-veriş, kampaniya və əlaqə tarixçəsi tək ekranda
  • Uçdan-uca loyallıq — POS terminal, onlayn satış və Mobil App ilə inteqrasiyalı vahid sadəqət proqramı
  • Seqmentasiya idarəetməsi — müştəri davranışlarını müəyyənləşdirin, hədəflənmiş təkliflər göndərin
  • Kampaniya idarəetməsi — mərkəzdən planla, filiallarda anında aktivləşdir

Süreç İdarəetməsi

Şirkətiniz təsdiq axınları ilə işləsin — siz harada olursunuz olun

Qiymət dəyişikliklərindən kampaniyalara, son istifadə tarixi (SKT) nəzarətindən sayımlara qədər sistemin tamamı süreç idarəetməsinə bağlıdır. Hər kritik əməliyyat təsdiq axınından keçir — ofisdə olmadan, telefonunuzdakı mobil tətbiqdən təsdiqləyin.

  • Qiymət dəyişikliyi, kampaniya, sayım və SKT nəzarəti — hamısı təsdiqə bağlı
  • Mobil tətbiqdən bildiriş → bir toxunuşla təsdiq və ya rədd
  • Səlahiyyət səviyyələrinə görə çoxpilləli təsdiq zəncirləri
  • Hər addım audit jurnalında — kim, nə vaxt, nəyi təsdiqlədi

📊 Bold BI · Power BI · Microsoft 365

BI və ofis alətlərinizlə birbaşa inteqrasiya: canlı data qoşulmaları, avtomatik hesabat paylaşımı.

🔄 Aparıcı ERP-lər və API

SAP, LOGO, Mikro, 1C, jRetail, Venüt, Microsoft Dynamics 365 — bazarın öndə gedən ERP məhsulları ilə dəyişkən data inteqrasiyası; MilliON kimi tətbiqlərlə API dəstəkli hazır bağlantılar.
